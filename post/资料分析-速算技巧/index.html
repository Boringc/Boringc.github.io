截几位？ 看选项差距

1.选项差距大（截==两位==）

四个选项==首位==均不同 选项1. A.65 B.53 C.47 D.38

首位相同，第二位差大于首位 选项2.A.65 B.53 C.59 D.47

2.选项差距小（截==三位==）

首位相同==且第二位差==小于等于==首位== 选项2. A.55 B.54 C.53 D.52

==99%的题目==都是直接截位

只有当选项之间存在约==10、100倍等==关系时，需要先考虑量级，再截位计算

速算本身是灵活的，对多步除法：

选项差距大：上下截两位后约分化简（可微调±1）
选项差距小：后面到基期比重时会有分析的办法

分数比较

分子分母一大一小

送分，直接看：==分子大的分数大（钱多人少）==

67\frac{6}{7}76​ 与 59\frac{5}{9}95​ 25331\frac{253}{31}31253​ 与 21542\frac{215}{42}42215​

分子分母同大同小

竖着直接除：一般分母截两位计算即可

横着看倍数：分子间倍数大，看分子，分子大的分数大 67\frac{6}{7}76​ 与 4956\frac{49}{56}5649​

​ 分母间倍数大，看分母，分母大的分数小 22423\frac{224}{23}23224​ 与 44047\frac{440}{47}47440​
